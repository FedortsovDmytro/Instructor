# Instructor

## Overview
**Instructor** is a Java-based web application that allows instructors to post, update, and delete courses, while students can read and enroll in courses. It demonstrates my skills in Java programming, object-oriented design, web application development, and database integration.

## Features
- Instructors can create, update, and delete courses.
- Students can view and enroll in courses.
- Secure authentication and role-based access control.
- Persistent data storage using MySQL.
- Dynamic web interface powered by Thymeleaf.

## Tech Stack
- **Language:** Java  
- **Database:** MySQL  
- **Frameworks / Libraries:** Spring Boot, Hibernate, JDBC, Thymeleaf, Spring Security  
- **Tools:** IntelliJ IDEA, Maven, Git/GitHub
